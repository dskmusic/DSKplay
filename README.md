# DSK Play

DSK Play is a mobile music streaming app built with Flutter, focused on a clean listening experience: search and stream music, manage playlists and liked songs, listen to radio stations, browse and play local audio files, and take your library offline.

## Features

- Search and stream music
- Playlists, liked songs and recently played
- Offline downloads
- Radio stations
- Local file browser and playback (folders, multi-select, queue integration)
- Synced (karaoke-style) lyrics with customizable colors and fullscreen mode
- Light/dark themes with dynamic color support

## Building

This is a standard Flutter project.

```bash
flutter pub get
flutter build apk --release
```
